# LUXE CASE - 프리미엄 폰케이스 브랜드 랜딩 페이지 PRD

## 프로젝트 개요
야놀자 그룹의 디자인 시스템을 참고하여 제작된 프리미엄 폰케이스 브랜드 'LUXE CASE'의 랜딩 페이지입니다.

## 목표
- 모던하고 프리미엄한 브랜드 이미지 구축
- 제품 라인업 효과적으로 전시
- 원활한 사용자 경험 제공
- GitHub Pages를 통한 배포

## 주요 기능

### 1. 히어로 슬라이더
- 3개의 풀스크린 슬라이드
- 자동/수동 네비게이션
- 키보드 제어 지원

### 2. 제품 컬렉션
- 4개의 제품 카드 그리드
- Hover 인터랙션
- 태그 시스템

### 3. 브랜드 스토리
- 2단 레이아웃
- 스크롤 애니메이션

### 4. 프리미엄 특징
- 6개의 특징 카드
- 아이콘 기반 UI

### 5. 통계 섹션
- 카운터 애니메이션
- 글로벌 진출 지표

### 6. 반응형 디자인
- 모바일 최적화
- 태블릿 지원
- 데스크탑 레이아웃

## 기술 스택
- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript

## 배포
- GitHub Pages
- 정적 사이트 호스팅

## 성공 지표
- 페이지 로딩 속도 < 2초
- 모든 디바이스에서 원활한 작동
- 접근성 준수
